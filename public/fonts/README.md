# Fonte Bellamora (principal do site)

A fonte **Bellamora** é a principal do site (display), alinhada ao logotipo.

Coloque aqui os arquivos da **Bellamora Free Personal Use**:

- `Bellamora-Free-Personal-Use.woff2` (recomendado para web)
- `Bellamora-Free-Personal-Use.woff` (fallback)

O `@font-face` em `styles/globals.css` usa esses nomes com hífens. Se seus arquivos tiverem outro nome, renomeie ou ajuste o CSS.

**Onde baixar:** versão gratuita para uso pessoal em [BeFonts](https://befonts.com/bellamora-font.html) ou similar.
